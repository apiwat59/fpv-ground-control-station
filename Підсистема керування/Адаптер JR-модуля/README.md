# Загальний опис

Адаптер JR-модуля забезпечує можливість підключення пульта керування з JR-сумісним відсіком до блоку керування станцією для подальшого двостороннього обміну інформацією з передавачем керування на виносному блоці станції.

<img width="800" alt="Загальний вигляд JR-модуля" src="picture/general_view_of_the_JR_module.png" />

## Схемотехніка та функціонал адаптера JR-модуля

Живлення адаптера здійснюється безпосередньо від пульта керування через роз’єм XS1 у діапазоні 5В–8,4В.

Високошвидкісний сигнал від пульта керування через вивід S.Port (протокол CRSF) роз’єму XS1 надходить на конвертер інтерфейсів (модуль BARVINOK-5 RS-485 nano V2), який перетворює його в диференційний сигнал стандарту RS-485 та через роз’єм XS2 видає в комутаційні лінії наземної станції. 

Стабілізацію температурних режимів пристрою забезпечує пасивна система охолодження, що складається з вентиляційних отворів у корпусі, силіконової термопрокладки та мідного радіатора. Радіатор електрично з’єднаний із загальним проводом (GND), що дозволяє йому виконувати функцію додаткового екрана для захисту від електромагнітних завад.
