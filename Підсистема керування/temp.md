# Загальний опис

Підсистема керування забезпечує двосторонній обмін даними через комутаційні лінії наземної станції за за стандартом RS-485 між пультом керування оператора та передавачем керування, що встановлений у виносному блоці наземної станції.

До складу підсистеми керування входять:
- Адаптер JR-модуля (зі сторони пульта керування)
- Блок TX (зі сторону виносного блоку)

## Короткі технічні параметри

| Параметр | Значення | Примітка |
|----------|---------|---------|
| Протокол керування | CRSF | Через S.Port |
| Інтерфейс передачі | Диференційний сигнал стандарту RS-485 | |
| Канал передачі даних | Комутаційні лінії наземної станції | |
| Режим роботи | Двосторонній | Керування + телеметрія |
| Живлення адаптера JR-модуля | 5–8.4 В | Від пульта керування |
| Живлення блоку TX | Від концентратора виносного блоку | Через XS2 |
| Вихідна напруга блоку TX | 8В | |
| Охолодження | Пасивне | Радіатори + вентиляційні отвори |
| Екранування | Часткове | |

## Принцип роботи та архітектура

Сигнал керування від пульта надходить на адаптер JR-модуля, де перетворюється в диференційний сигнал стандарту RS-485. Далі сигнал передається через комутаційні лінії наземної станції до блоку TX, де відбувається зворотне перетворення в сигнал протоколу CRSF перед подачею на на передавач керування. Зворотний канал (телеметрія) працює аналогічно у протилежному напрямку.

## Примітка

Детальна реалізація кожного пристрою підсистеми керування наведена у відповідних розділах:
- Адаптер JR-модуля
- Блок TX
