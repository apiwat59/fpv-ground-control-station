# Загальний опис

Універсальна рамка призначена для монтажу та механічної інтеграції концентратора виносного блоку та периферійних пристроїв. Концентратор виносного блоку разом з встановленими на універсальній рамці периферійними пристроями формує виносний блок наземної станції.

Конструкція рамки забезпечує:
- встановлення концентратора виносного блоку
- встановлення блоку TX та VRX блоків
- уніфіковане кріплення компонентів системи
- швидке переведення виносного блоку між транспортним та робочим положенням

## Особливості конструкції

- Конструкція має модульну архітектуру
- Передбачено встановлення змінних блоків TX та VRX блоків
- Всі модулі фіксуються гвинтовими з’єднаннями
- Конструкція забезпечує стійкість до механічних навантажень та вібрацій

## Транспортне положення

Транспортне положення виносного блоку в корпусі наземної станції передбачає:
- демонтовану підставку
- відсутність VRX блоку в додатковому слоті
- фіксоване положення периферійних пристроїв
