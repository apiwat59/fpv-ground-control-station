# Загальний опис

Кронштейн призначений для монтажу виносного блоку наземної станції керування на щоглі або інших несучих конструкціях. Кронштейн є частиною системи механічної інтеграції виносного блоку. Конструкція орієнтована на польову експлуатацію та транспортування. Загальний вигляд виносного блоку з додатковими антенами закріпленого через кронштейн на щоглі показано на картинці.

<img width="800" alt="Загальний вигляд виносного блоку з додатковими антенами на щоглі" src="picture/general_view_of_the_remote_unit_on_the_mast.png" />

Конструкція кронштейна забезпечує:
- швидке розгортання
- фіксацію виносного блоку
- монтаж штанг для встановлення додаткових антен чи обладнання
- збільшення висоти встановлення антен
- розширення антенної конфігурації відповідно до умов експлуатації
